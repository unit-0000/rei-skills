# rei-skills

Agent skills for [Rei Labs](https://reilabs.org).

## Install

```bash
npx skills add 00unitrei/rei-skills
```

## Skills

- **[opencode-reilabs-provider](skills/opencode-reilabs-provider/SKILL.md)** — Install the Rei Labs provider into OpenCode so the `rei-qwen3-coder`, `rei-coder-lite`, `rei-coder-pro`, and `rei-coder-max` models become available.
